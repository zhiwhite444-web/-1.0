# SillyTavern GrokChat Theme

Grokchat 风格的 SillyTavern 主题扩展。

## 文件结构

```
sillytavern-grokchat-theme/
├── manifest.json
├── scripts/
│   └── grokchat-overlay.js
└── styles/
    └── grokchat-theme.css
```

## 安装

复制此文件夹到 SillyTavern 扩展目录：
```
~/.config/SillyTavern/data/default-user/extensions/sillytavern-grokchat-theme/
```

## 使用

1. 重启 SillyTavern
2. 鼠标悬停在左侧区域展开侧边栏
3. 鼠标离开后收起
